Pinweatherfy

                            AdR - Grupo 2

Descripción

Este proyecto se basa en la combinación de las aplicaciones Spotify, Pinterest y Openweathermap.
Su objetivo es el de proveer de atuendos o inspiraciones para la vestimenta del usuario en función del tiempo climático, así como de una lista de reproducción de Spotify acorde.
