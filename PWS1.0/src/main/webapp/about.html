Sobre Nosotros

Descripción

Este proyecto se basa en la combinación de las aplicaciones Pinterest, OpenWeatherMap y Spotify.

Su objetivo es el de proveer de autendos o inspiraciones para la vestimenta del usuario en función del clima, así como una lista de reproducción de Spotify acorde.
